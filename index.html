서버 소개

유튜버 스토리님의 공식 디스코드 서버입니다. 서버 초대 링크는 아래 링크만을 사용합니다.
https://discord.gg/KYWs4rKfVw

가입 조건

  1. 스토리님의 팬이어야 합니다.
  2. 디스코드 계정에 휴대폰인증이 되어있어야 합니다.

채널 목록

  • #규칙
    규칙을 안내하고 동의를 받는 채널입니다.
  • #잡담, #잡담-OO관련
    해당 주제에 관련된 대화를 자유롭게 나누는 곳입니다.
  • #스토리님의-공지, #서버장의-공지
    각각 스토리님과 서버장이 공지사항을 게시하는 채널입니다.
  • #유튜브
    새로운 유튜브 영상이 올라왔을 때 알려줍니다. (놀랍게도 거의 항상 유튜브 자체 알림보다 빨리 알려줍니다!)
  • #서버현황 카테고리
    서버의 현재 멤버 수나 다른 현황 수치를 보여줍니다.
  • #스토리님께
    스토리님께 남기고 싶은 말을 쓸 수 있습니다.
  • #상담신청창구
    관리진과 면담을 신청할 수 있습니다.
  • #랭크확인
    자신의 미육 랭크를 확인하는 곳입니다. 전체 랭킹순위는 여기에서 확인하세요.
  • #사진
    모든 사진들이 보관되는 곳입니다.
  • #신고함
    부적절한 행동을 신고하는 곳입니다.

봇 소개

보토리 Botory

스토리 서버에서 많은 일들을 도맡아 하는 봇으로, 서버장이 직접 개발하였습니다. 보토리는 주로 다음과 같은 일들을 합니다.

  • 채팅 줄 수, 글자 수 제한
  • 채팅 및 반응의 중지 이모지 검열
  • 사진 크기 축소 (도배 방지)
  • 반응 로그 기록
  • 자동 이모지 반응 (소위 ‘펫’)
  • 서버 멤버 수 측정
  • 기타 등등

미육 MEE6

랭킹 시스템과 경고 시스템을 담당합니다.

그 외

  • 로거 Logger
    서버의 거의 모든 활동을 기록합니다.
  • 티켓툴 Ticket Tool
    상담신청 창구 혹은 서버장 호출 상담을 할 때 비공개 채널을 생성하고 관리해줍니다.
  • 마크 봇
    마인크래프트 서버 상태 알림 등을 합니다.
  • 그루비 Groovy
    음악 봇입니다. 거의 쓰지 않습니다.

관리직 소개

관리직 공채는 스토리님이나 서버장이 따로 공지하지 않는 이상 절대 추가 공채하지 않습니다.
관리자가 되고 싶다거나 관리자를 더 뽑자는 등의 발언 하지 말아주세요.

경찰

  • 파란색 닉네임과 [경찰]접두사를 보유
  • 검찰이 되기 위한 견습생 단계
  • 악질 분탕이 발생했을 때 채팅중인 관리자가 보이지 않는다면 @경찰로 호출하시면 됩니다.
  • 뮤트 권한 보유

검찰

  • 주황색 닉네임과 [검찰]접두사를 보유
  • 경찰이 가진 모든 권한 보유
  • 경고 부여 권한 보유

의원

  • 노란색 닉네임과 [의원]접두사를 보유
  • 검찰이 가진 모든 권한 보유
  • 슬로우 모드 조정 권한 보유

규칙

스토리 서버에는 엄격한 규칙들이 있습니다. 아래 규칙들을 잘 숙지하여 불이익을 받지 않도록 주의하세요.

처벌 시스템

  • 스토리 서버는 미육 경고 시스템을 활용합니다.
  • 용어 정리
    • 경고 : 말 그대로 경고입니다. 한 두번의 경고는 활동에는 지장이 없습니다.
    • 뮤트 : 채팅 금지입니다. 뮤트되면 채팅을 칠 수 없습니다.
    • 킥 : 서버에서 추방되는 것입니다. 다시 들어올 수는 있습니다.
    • 밴 : 서버에서 추방되고 다시 들어오지 못하게 되는 것입니다.
  • 관리자는 두배의 처벌을 받습니다.
  • 30분 내로 경고 3회 누적시 미육에 의해 자동으로 2시간 뮤트됩니다.
  • 24시간 내로 경고 5회 누적시 미육에 의해 자동으로 24시간 뮤트됩니다.

경고 대상

  1. 정치질
    (“검거” “규칙좀 읽으세요 ㅉㅉ” 등을 말합니다. 타인이 규칙을 어기면 조용히 신고만 하세요.)
  2. 고의로 채널 주제에 맞지 않는 채팅
  3. 슬로우 모드에 대한 투정
  4. 허가 없는 외부활동 모집과 홍보
  5. 다른 사람에게 반말 (음슴체 포함)
  6. 도배 혹은 도배를 유도하려는 의도가 명확한 채팅
  7. 노체 혹은 누체 (관련 밈 포함)
  8. 욕설
  9. 외부 링크 첨부
  10. 8개 이상의 과도한 이모지를 쓴 채팅
  11. 3개 이상의 과도한 멘션이나 스포일러 태그
  12. zalgo 사용
  13. 의미없는 채팅 반복
    (이 경우 랭크도 초기화 조치합니다.)
  14. 특정 사용자들끼리 친목
  15. 경고 기록 확인 요청
  16. 규칙 문서를 읽지 않고 규칙에 대해 질문
  17. 멤버들간에 DM
  18. 서버장을 멘션
  19. !rank이외의 명령어 사용 시도
  20. FAQ에 있는 내용 질문
  21. 100만 이하 타유튜버 밈이나 언급
    (스토리님이나 서버장이 먼저 대화 주제를 꺼낸 경우는 예외입니다.)

밴 대상

  1. 정치적 발언이나 정치적 인물이 포함된 사진, 정치 밈 등 모든 정치적 어필
  2. 홍보
  3. 일베
  4. 분탕
  5. 그 외 모든 고의로 타인을 기분나쁘게 하는 행동

그 외 지침

  1. 웬만하면 멘션을 사용하지 말아주세요.
  2. 구글에 검색해서 알아낼 수 있는 내용을 질문하여 분위기를 흐리지 말아주세요.
  3. 닉네임이나 상태메시지에 욕설이나 부적절한 어휘를 사용한 경우 닉네임을 zz검열 꼴로 강제 변경합니다.
  4. 갈고리 물타기 등 물타기가 너무 많이 일어난다고 생각되면 알아서 자제해주세요.
